# DOI-Wallet-iX – Schnellstart-Anleitung

**Version 0.9.7 (Beta)**

---

## Schritt 1: Start

Starten Sie `DOI-Wallet-iX.exe`.

## Schritt 2: Wallet erstellen oder öffnen

- **Neues Wallet:** „Neues Wallet erstellen", 24 Wörter auf Papier notieren,
  Passwort vergeben.
- **Bestehendes Wallet:** „Wallet öffnen" und Passwort eingeben.
- **Wiederherstellen:** „Wallet wiederherstellen (Seed)", 24 Wörter eingeben.
  Falls Sie eine zusätzliche BIP-39-Passphrase verwenden, diese ebenfalls
  angeben.

## Schritt 3: Dashboard

Salden, Adressen und Verbindungsstatus auf einen Blick.

## Schritt 4: Empfangen

Tab „Empfangen": Adressen mit QR-Codes. Klick zum Kopieren. Bei DOI wird bei
jedem Aufruf eine frische Adresse erzeugt – alte Adressen bleiben gültig.

## Schritt 5: Senden

Tab „Senden": Währung, Adresse und Betrag eingeben, mit dem
**Wallet-Passwort** bestätigen (wird seit v0.9.7 wirklich geprüft, dauert
1–2 Sekunden). Bei Erfolg erscheint die Transaktions-ID, bei Fehlern eine
klare Fehlermeldung.

## Schritt 6: Weitere Wallets (optional)

Klick auf leeren Tab oben für ein weiteres Wallet. Doppelklick zum
Umbenennen.

## Schritt 7: Sicherheit konfigurieren (empfohlen)

Unter „Einstellungen" können Sie ein Tageslimit in EUR setzen und den
Undo-Timer aktivieren (10–60 Sekunden zum Abbrechen nach dem Senden).

**Tipp:** Tragen Sie dort auch einen kostenlosen TronGrid-API-Key ein
(<https://www.trongrid.io>) – das verhindert „Rate-Limit"-Fehler beim Senden
von TRX/USDT.

## Schritt 8: Transaktions-Notizen

Klicken Sie auf eine Transaktion in der Transaktionsliste, um eine
persönliche Notiz hinzuzufügen (z. B. „Bezahlung für Webseite"). Klick auf
den TX-Hash kopiert die vollständige Transaktions-ID.
